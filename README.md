# Custom-Libraries

A collection of reusable C++ header libraries built from scratch.

## 📚 Libraries

| Library | Description |
|---------|-------------|
| **clsDate.h** | Date and time operations |
| **ClsPeriod.h** | Period calculations between dates and times |
| **ClsString.h** | String manipulation and utility functions |
| **ClsPerson.h** | Person data management (name, address) |
| **clsInputValidate.h** | Input validation for numbers and screen reading |
| **ClsUtil.h** | General utilities including random number generation, random words/keys, array filling, and swap operations |
